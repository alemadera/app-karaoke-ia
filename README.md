# 🎤 App Karaoke con IA

Esta aplicación permite separar voces de un video (Demucs) y generar subtítulos automáticamente (Whisper).

## Instrucciones

1. Sube un video 🎥
2. Separa las voces (IA)
3. Genera subtítulos (opcional)
4. Obtén un video estilo karaoke 🎶

Desarrollado con Python + Streamlit + Whisper + Demucs.
